# bashScripts
## update_git_repos.sh
Copy the file to the dir/folder where your list of the repos are storing.
chnage the current location to the target dir/folder
for example:
```
-<GitStorage>
--<repo1>
--<repo2>
--...
```
run:
```
<user>@<host>:/path/to/GitStorage$ ./update_git_repos.sh master
```
